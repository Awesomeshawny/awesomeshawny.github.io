Clone of the Dead

An on-rails shooter inspired by The House of the Dead.

Overview

Clone of the Dead is a zombie shooter inspired by 90s lightgun arcade games.
The project spanned 2 weeks and required a pitch, minimum viable product, and final prototype utilizing the Unity Asset Store.

Game Design, Prototyping, Modular Levels

Key Features

Modular Level Design

Levels are built with easily configurable components.

Enemy AI

Behaviours driven by a Finite State Machine.

Spawning

The environment adapts according to audience votes.

Dynamic Prompt Selection

Randomized prompt generation ensures each performance is unique while maintaining structured pacing and fairness.
